
# TodoApp Mobile 📱


![Animação](https://user-images.githubusercontent.com/91674510/210689243-4957b83d-3099-4d43-b80e-50abd7fa534e.gif)



## Projeto 🧑🏻‍💻

Que tal um aplicativo que vai listar todos as suas tarefas no dia a dia ? Esse aplicativo vai lhe ajudar com suas tarefas do seu dia.


## Tecnologias Usadas 🏗️
- React Native
- Typescript
- Expo
## Instalação 🚀

```bash
  npm install 
  npm start 
```
Com isso vai ser gerado o link ou qrCode para acessar através do aplicativos Expo Go que tem nas lojas de aplicativos do seu celular.

## Funcionalidades 🌟
📌 Cadastro de tarefas  
📌 Deletação de tarefas  
📌 Consulta de quantidades de tarefas
